## Project Overview

This MERN app serves as a **social media/blog platform** backend and frontend, with features for managing posts and users. It includes:

- RESTful API endpoints
- User authentication 
- CRUD operations on posts
- Frontend with React and Vite
- Real-time features via Socket.io 

---

## Features

- **User Management:** Create, update, delete, and retrieve user profiles  
- **Post Management:** Create, read, update, delete posts  
- **Real-Time Updates:** Receive live updates via WebSockets  
- **Health Check:** API status endpoint  
- **Testing:** Unit, integration, and E2E tests using Jest, Supertest, and Cypress

---

## Tech Stack

- **Backend:** Node.js, Express.js, MongoDB, Mongoose, Socket.io  
- **Frontend:** React.js, Vite, Axios  
- **Testing:** Jest, Supertest, Cypress  
- **Environment:** Node.js >= 18, Render (deployment)
